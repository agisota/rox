# Oh My Codex (OMX)

Orchestration layer for OpenAI Codex CLI. Like oh-my-zsh but for Codex.

## Description
Adds hooks, agent teams, HUDs, autonomous workflows, and structured modes to Codex CLI.

## Source
https://github.com/Yeachan-Heo/oh-my-codex

## Key Features
- Async Claude Code delegation (no timeouts)
- Structured workflows: autopilot, TDD, code review, planning
- Persistent state & memory
- Team mode and swarm coordination

## Included Sub-Skills
- omx-autopilot, omx-team, omx-swarm
- omx-tdd, omx-code-review, omx-security-review
- omx-ralph, omx-ralplan, omx-ultrawork
- And 30+ more

# Grillme

Use the installed `grill-me` skill.

Interview the user or the available evidence until the plan, design, or decision
tree is explicit. Resolve dependencies one branch at a time. If the answer can
be discovered from the codebase or local context, inspect that evidence instead
of asking.

Ask only one unresolved question at a time, and provide the recommended answer
when the evidence supports one.

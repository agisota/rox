Rox — это платформа для AI-программирования с открытым исходным кодом. Она доступна как десктопная IDE, интерфейс командной строки или MCP-сервер.

<YouTubeVideo id="7jhPfMDwTUc" title="Rox Demo" thumbnail="/images/video-thumbnail.png" />

Каждая ветка git получает собственное изолированное рабочее пространство через git worktrees — без stash, без резких переключений `git checkout`. Запускайте Claude Code, Cursor и других агентов параллельно в разных рабочих пространствах, открывайте любое из них в вашей IDE и просматривайте/коммитьте изменения во встроенном просмотре изменений (diff viewer). Локально-ориентированный подход: работает офлайн и синхронизируется при подключении.

## Требования

- macOS (Apple Silicon или Intel) — поддержка Windows и Linux появится скоро
- Установленный Git
- Аутентифицированный GitHub CLI (`gh auth status`)

## Добавление репозитория

Добавьте свой репозиторий из локальной папки или по Git URL. Требуется аутентификация GitHub (`gh auth status`).

![Add Repository](/images/add-repo.png)

## Создание рабочего пространства

Каждое рабочее пространство — это изолированный git worktree на собственной ветке. Rox копирует только файлы, отслеживаемые git — используйте [скрипты настройки](/setup-teardown-scripts) для файлов `.env` или `bun install`.

Создавайте рабочие пространства из:
- Новой ветки от main
- Существующей ветки
- Pull request

В сценарии «сначала промпт» вы можете привязать pull request, выполнив поиск по нему или вставив полный URL PR.

![New Workspace](/images/new-workspace.png)

## Начало работы

- **⌘O** — открыть в Cursor/VS Code
- **Terminal** — встроенный терминал готов к использованию
- **Changes** — просмотр изменений и коммит

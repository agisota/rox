## Обзор

Rox работает с монорепозиториями. Рабочие пространства создаются в корне репозитория с доступом ко всем пакетам.

## Скрипт настройки

```json
{
  "setup": [
    "bun install",
    "cp \"$ROX_ROOT_PATH/.env\" .env"
  ]
}
```

## Запуск сервисов

Запускайте несколько сервисов из терминала:

```bash
bun run dev                          # All services
bun run --filter @myapp/web dev &    # Specific package
bun run --filter @myapp/api dev
```

## Git Submodules

Добавьте в скрипт настройки:

```json
{
  "setup": ["git submodule update --init --recursive", "bun install"]
}
```

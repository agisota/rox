Вы можете импортировать файл темы из Marketplace или создать свой на основе базового файла в настройках десктопного приложения.

## Быстрый процесс

1. Откройте `Settings → Appearance → Theme`.
2. Либо нажмите `Marketplace`, чтобы скачать опубликованную тему, либо нажмите `Download Base File`, чтобы начать с вашей текущей темы.
3. Если вы скачали базовый файл, отредактируйте JSON вручную или попросите агента настроить его.
4. Нажмите `Import Theme` и выберите файл.
5. Выберите импортированную тему из сетки тем.

Темы из Marketplace доступны на странице <a href={`${COMPANY.MARKETING_URL}/marketplace/themes`}>Marketplace</a> или по ссылке `Marketplace` в `Settings → Appearance → Theme`.

## Попросите агента настроить тему

После загрузки базового файла вы можете дать агенту запрос вроде такого:

```text
Customize this Rox theme JSON to feel like a neon cyberpunk arcade.
Keep the schema valid. Preserve all required ui and terminal keys.
Return the full JSON file contents only.
```

Затем сохраните полученный JSON и импортируйте его в Rox.

## Формат файла темы

Rox принимает:

- Один объект темы
- Массив объектов тем
- Объект с `themes: [...]`

Импортёр поддерживает:

- `id`, `name`, `type` (`dark` или `light`)
- переопределения цветов `ui`
- переопределения цветов `terminal`
- `colors` как псевдоним для `terminal`

Отсутствующие значения заменяются значениями по умолчанию Rox для выбранного типа темы.

## Пример: одна тема

```json
{
  "id": "my-neon-theme",
  "name": "My Neon Theme",
  "type": "dark",
  "author": "You",
  "description": "Neon terminal and UI accents",
  "ui": {
    "background": "#12061f",
    "foreground": "#f7f5ff",
    "primary": "#39ff14",
    "accent": "#00e5ff",
    "border": "#7b2dff"
  },
  "terminal": {
    "background": "#0b0314",
    "foreground": "#f8f2ff",
    "cursor": "#39ff14",
    "red": "#ff2b70",
    "green": "#39ff14",
    "yellow": "#ffef00",
    "blue": "#00b7ff",
    "magenta": "#ff5ecf",
    "cyan": "#00ffd1",
    "white": "#e9dcff",
    "black": "#18082a",
    "brightBlack": "#6e3ea5",
    "brightRed": "#ff6fa2",
    "brightGreen": "#7dff5a",
    "brightYellow": "#fff86a",
    "brightBlue": "#59dbff",
    "brightMagenta": "#ff9be8",
    "brightCyan": "#8dffe8",
    "brightWhite": "#ffffff"
  }
}
```

## Примечания

- Идентификаторы тем должны быть уникальными и не зарезервированными (`system`, встроенные ID тем вроде `dark`/`light`).
- Очень большие файлы отклоняются импортёром десктопного приложения.

## Обзор

Rox включает встроенную панель браузера, так что вы можете предпросматривать запущенные сервисы, не покидая приложение.

## Открытие вкладки браузера

- Нажмите **⌘⇧B**, чтобы открыть новую вкладку браузера
- Кликните по иконке внешней ссылки на любом обнаруженном порте в боковой панели, чтобы открыть его в браузере

## Навигация

Панель браузера включает полную адресную строку с:

- Кнопками «Назад», «Вперёд» и «Перезагрузить»
- Автодополнением URL на основе вашей истории просмотров
- Отображением favicon для посещённых сайтов

## История просмотров

Посещённые вами URL сохраняются локально и появляются как предложения автодополнения. Чтобы очистить историю:

1. Кликните по меню дополнительных действий (⋯) на панели инструментов браузера
2. Выберите **Clear Browsing History**

## DevTools

Откройте панель DevTools рядом с любой вкладкой браузера, чтобы инспектировать и отлаживать страницу, запущенную в браузере.

## Интеграция с портами

Когда Rox обнаруживает используемый порт (например, `localhost:3000`), клик по его иконке внешней ссылки открывает его во вкладке встроенного браузера вместо системного браузера.

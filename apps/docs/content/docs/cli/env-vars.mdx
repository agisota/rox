title: Переменные окружения
description: Полный справочник по переменным окружения, управляющим поведением Rox CLI.
---

Rox CLI поддерживает следующие переменные окружения для управления
своим поведением. Задайте их в оболочке перед запуском `rox` или
экспортируйте из профиля оболочки (`~/.zshrc`, `~/.bashrc`) либо из
окружения CI, чтобы применить их ко всем сессиям.

| Переменная | Описание |
| --- | --- |
| `ROX_API_KEY` | Ключ API (`sk_live_…` / `sk_test_…`), используемый вместо сохранённого входа через OAuth. Эквивалент `--api-key`. |
| `ROX_HOME_DIR` | Каталог, который CLI использует для `config.json` и дерева хост-сервера. По умолчанию: `~/.rox`. Общий с десктоп-приложением. |
| `GH_TOKEN` / `GITHUB_TOKEN` | Читается через `gh` CLI, к которому хост-сервер обращается для checkout PR и метаданных. Задайте эту переменную для песочниц или CI вместо интерактивного запуска `gh auth login`. |

Скрипт автономной установки также читает:

| Переменная | Описание |
| --- | --- |
| `ROX_HOME` | Корневой каталог установки автономного CLI и бинарного файла хоста. По умолчанию: `~/rox`. |
| `ROX_VERSION` | Тег релиза для установки. По умолчанию: `latest`, который разрешается через скользящий релиз `cli-latest`. |

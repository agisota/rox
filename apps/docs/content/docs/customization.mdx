## Звуки уведомлений

Получайте звуковые уведомления, когда агенты завершают работу или терминалы закрываются.

Settings → Notifications → выберите звук

Импортируйте собственные аудиофайлы рингтонов (MP3, WAV, OGG, до 20 МБ) в **Settings → Notifications**. Прослушайте любой импортированный звук и установите его в качестве активного уведомления.

## Цвета проектов

Выделяйте проекты цветом в боковой панели для быстрой идентификации.

Нажмите настройки проекта → выберите цвет

## Редактор по умолчанию

Задайте, какой редактор открывается при нажатии ⌘O.

Settings → Editor → выберите Cursor, VS Code или другой

## Режим открытия файлов

Управляйте тем, как открываются файлы при клике в дереве файлов или представлении изменений.

Settings → Behavior → File open mode

- **Split pane** (по умолчанию) — открывает файлы, разделяя текущую вкладку
- **New tab** — открывает файлы в новой вкладке

Повторное использование панели предпросмотра работает в обоих режимах — эта настройка влияет только на запасной вариант, когда нет переиспользуемой панели.

## Пресеты терминала

Сохраняйте конфигурации терминала (команды, окружение) как пресеты для быстрого доступа.

Settings → Terminal → Manage Presets

## Пользовательские темы

Скачайте тему из Marketplace или начните с базового JSON темы и настройте её перед импортом.

Settings → Appearance → Theme → Marketplace / Download Base File / Import Theme

Полное руководство: [Пользовательские темы](/custom-themes)

## Синхронизация настроек

Ваши настройки синхронизируются между устройствами при входе в аккаунт.

## Обзор

Просматривайте изменения, индексируйте файлы, коммитьте и отправляйте — всё из одного интерфейса.

![Diff Viewer](/images/changes.png)

## Возможности

- **View changes** — просмотр изменённых, проиндексированных и неотслеживаемых файлов
- **Stage/unstage** — выбор файлов для коммита
- **Commit** — написание сообщения и коммит
- **Push/Pull** — синхронизация с удалённым репозиторием
- **Create PR** — открытие pull request на GitHub

## Раскладки diff

- **Split view** — сравнение бок о бок
- **Unified view** — изменения в одну колонку

## Выделение текста

Выделяйте текст прямо в diff и копируйте его из контекстного меню.

Дважды кликните по строке diff или кликните правой кнопкой и выберите **Edit Here**, чтобы открыть исходный редактор в соответствующем месте.

## Режим фокусировки

Просматривайте по одному файлу за раз с элементами навигации.

Включите режим фокусировки на панели инструментов diff (иконка цели), чтобы:
- Просматривать один файл с минимумом отвлекающих факторов
- Перемещаться между файлами с помощью кнопок Prev/Next
- Переходить между разделами (Against base, Commits, Staged, Unstaged) через выпадающий список разделов
- Видеть количество файлов и позицию по каждому разделу (например, «2/5 in Staged»)

Кликайте файлы в боковой панели, чтобы сразу перейти к ним в режиме фокусировки.

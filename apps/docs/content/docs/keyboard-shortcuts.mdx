## Рабочее пространство

| Действие | Сочетание |
|--------|----------|
| Новое рабочее пространство | ⌘N |
| Быстрое создание рабочего пространства | ⌘⇧N |
| Открыть проект | ⌘⇧O |
| Переключиться на рабочее пространство 1–9 | ⌘1-9 |
| Предыдущее рабочее пространство | ⌘⌥↑ |
| Следующее рабочее пространство | ⌘⌥↓ |

## Раскладка

| Действие | Сочетание |
|--------|----------|
| Переключить вкладку изменений | ⌘L |
| Развернуть/свернуть боковую панель | ⌘⇧L |
| Переключить боковую панель рабочих пространств | ⌘B |
| Разделить вправо | ⌘D |
| Разделить вниз | ⌘⇧D |
| Авторазделение панели | ⌘E |
| Закрыть панель | ⌘W |
| Быстрое открытие файла | ⌘P |
| Поиск по ключевым словам | ⌘⇧F |
| Новая вкладка браузера | ⌘⇧B |

## Чат

| Действие | Сочетание |
|--------|----------|
| Найти в чате | ⌘F |
| Фокус на поле ввода чата | ⌘J |

## Терминал

| Действие | Сочетание |
|--------|----------|
| Новая вкладка | ⌘T |
| Закрыть вкладку | ⌘⇧W |
| Закрыть терминал | ⌘W |
| Очистить терминал | ⌘K |
| Найти в терминале | ⌘F |
| Прокрутить вниз | ⌘⇧↓ |
| Предыдущая вкладка | ⌘⌥← |
| Следующая вкладка | ⌘⌥→ |
| Предыдущая панель | ⌘⇧← |
| Следующая панель | ⌘⇧→ |
| Открыть пресет 1–9 | ⌘⇧1-9 |

## Окно

| Действие | Сочетание |
|--------|----------|
| Открыть в приложении | ⌘O |
| Скопировать путь | ⌘⇧C |
| Закрыть окно | ⌘⇧Q |
| Показать горячие клавиши | ⌘/ |

## Настройка

Откройте Settings → Keyboard Shortcuts, чтобы настроить любую горячую клавишу.

## Импорт/экспорт

Поделитесь своей конфигурацией горячих клавиш:
- **Export** — сохраните горячие клавиши в файл
- **Import** — загрузите горячие клавиши из файла

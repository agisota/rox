## Общее

**Какие платформы?**
Только macOS. Windows/Linux появятся скоро.

**Как это работает?**
Использует git worktrees, чтобы у каждой ветки был свой каталог.

**Нужен ли аккаунт?**
Да, для командных функций и синхронизации настроек.

## Рабочие пространства

**Одна ветка в нескольких рабочих пространствах?**
Нет. Одна ветка = одно рабочее пространство. При необходимости создайте новую ветку.

**Где хранятся рабочие пространства?**
В каталоге данных Rox в виде git worktrees.

## Терминал

**Какой shell?**
Системный по умолчанию (bash/zsh) с вашими файлами конфигурации.

**Сохраняются ли сессии?**
Да, терминалы переживают перезапуски приложения.

## Git

**Изменяет ли это конфигурацию git?**
Нет. Используются стандартные команды git.

**Приватные репозитории?**
Да. Настройте авторизацию GitHub: `gh auth status`

## Открыть в редакторе

Нажмите **⌘O**, чтобы открыть рабочее пространство в вашем редакторе. Поддерживаемые редакторы:

- Cursor
- VS Code
- VS Code Insiders
- Zed
- Windsurf
- Antigravity
- Sublime Text
- JetBrains IDE (IntelliJ IDEA, WebStorm, PyCharm и другие)

![Open In Editor](/images/open-in.png)

## Настройка редактора по умолчанию

Settings → Editor → выберите предпочитаемый редактор

## Совет по заголовку окна

Отображение имени ветки в заголовке редактора:

1. Откройте настройки VS Code/Cursor
2. Найдите `window.title`
3. Установите значение: `${activeRepositoryBranchName}${separator}${rootName}`

Это помогает определить, какое окно есть какое, когда у вас открыто несколько рабочих пространств.

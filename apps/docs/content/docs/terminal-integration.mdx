## Обзор

Встроенный терминал для каждого рабочего пространства с несколькими вкладками и сохранением сессий.

![Terminal](/images/terminal.png)

## Несколько вкладок

Открывайте несколько вкладок терминала в каждом рабочем пространстве. Каждая работает независимо.

- **⌘T** — новая вкладка
- **⌘W** — закрыть вкладку
- **⌘1-9** — переключение между вкладками

## Сохранение сессий

Сессии переживают перезапуски приложения:
- Запущенные процессы продолжают работу
- История вывода сохраняется
- Прокрутка истории (scrollback) сохраняется

## Переименование панели

Дважды кликните по заголовку панели, чтобы переименовать её.

## Объединение вкладок

Перетащите вкладку в зону разделения другой вкладки, чтобы объединить панели исходной вкладки с этой раскладкой. Исходная вкладка закрывается после объединения.

## Меню по правому клику

Кликните правой кнопкой в терминале для быстрых действий:

- **Copy** — копировать выделенный текст (активно, когда текст выделен)
- **Paste** — вставить из буфера обмена (активно, когда в буфере есть содержимое)
- **Mark as Unread** — снова показать индикатор непрочитанного
- Split, Clear, Move to Tab, Close

## Ссылки по Cmd/Ctrl-клику

Удерживайте `Cmd` (macOS) или `Ctrl` и кликайте по ссылкам в выводе терминала:
- URL открываются в браузере
- Пути к файлам по умолчанию открываются во встроенном просмотрщике файлов
- Измените поведение для путей к файлам в **Settings → Terminal**

## Переменные окружения

| Переменная | Описание |
|----------|-------------|
| `ROX_ROOT_PATH` | Путь к корневому репозиторию |
| `ROX_WORKSPACE_NAME` | Имя текущего рабочего пространства |

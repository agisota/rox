## Вкладка ревью в сайдбаре изменений <PRBadge url="https://github.com/agisota/set/pull/2681" />

В сайдбаре изменений теперь есть отдельная вкладка «Ревью». Ты можешь читать, фильтровать и отвечать на комментарии ревью PR, не покидая десктопное приложение.

- Решённые комментарии отделены от открытых, чтобы ты мог сосредоточиться на том, что ещё требует внимания
- Остаётся синхронизированным с GitHub — автоматически обновляется после изменений PR

## Создание задач и действия с деталями <PRBadge url="https://github.com/agisota/set/pull/2656" /> <PRBadge url="https://github.com/agisota/set/pull/2660" />

![Модальное окно создания задачи](/changelog/create-task.png)

Ты теперь можешь создавать задачи и выполнять с ними действия прямо из десктопа, не переключаясь на Linear или веб-приложение.

- Новое модальное окно создания задачи для быстрого заведения задач откуда угодно
- Меню действий в деталях задачи для быстрых действий с отдельными задачами

## Настраиваемые пресеты уровня проекта <PRBadge url="https://github.com/agisota/set/pull/2719" />

![Пресеты уровня проекта](/changelog/project-presets.png)

Пресеты терминала теперь можно привязать к конкретному проекту. Пресеты уровня проекта появляются только когда ты работаешь в этом проекте, сохраняя список пресетов чистым и релевантным контексту.

## Кнопка запуска и команда остановки <PRBadge url="https://github.com/agisota/set/pull/2680" /> <PRBadge url="https://github.com/agisota/set/pull/2740" />

![Кнопка запуска](/changelog/run-button.png)

Кнопка запуска переехала на панель пресетов (`Cmd+G`) и теперь включает команду остановки, чтобы ты мог отменить запущенный пресет прямо во время выполнения.

- `Cmd+G` запускает пресет из любого места рабочего пространства
- Кнопка остановки сразу отменяет выполняющиеся запуски

## Улучшения

- **Имена веток, сгенерированные AI** — Rox теперь предлагает имя ветки при создании рабочего пространства с автоматическим разрешением конфликтов, если имя занято <PRBadge url="https://github.com/agisota/set/pull/2627" />
- **Черновик чата сохраняется** — неотправленный текст чата теперь сохраняется при переключении вкладок или рабочих пространств и восстанавливается, когда ты возвращаешься <PRBadge url="https://github.com/agisota/set/pull/2631" />
- **Предупреждение перед закрытием изменённых редакторов** — десктоп теперь спрашивает перед закрытием вкладки файла с несохранёнными изменениями <PRBadge url="https://github.com/agisota/set/pull/2717" />
- **Прикрепление issue GitHub в модальном окне рабочего пространства** — ты можешь прикрепить issue GitHub прямо в модальном окне нового рабочего пространства, давая агенту сразу контекст <PRBadge url="https://github.com/agisota/set/pull/2678" />
- **Автосохранение скриптов setup/teardown** — скрипты setup и teardown теперь сохраняются автоматически по мере того, как ты их редактируешь <PRBadge url="https://github.com/agisota/set/pull/2644" />
- **Перезагрузка по CMD+R теперь настраивается** — горячую клавишу перезагрузки `Cmd+R` можно переназначить или отключить в настройках клавиатуры <PRBadge url="https://github.com/agisota/set/pull/2716" />
- **Виртуализированная история git** — список коммитов в панели истории git теперь виртуализирован и ограничен 500, что предотвращает замедления в репозиториях с длинной историей <PRBadge url="https://github.com/agisota/set/pull/2763" />
- **Индикация загрузки при мерже PR** — кнопка мержа теперь показывает состояние загрузки, пока идёт мерж PR <PRBadge url="https://github.com/agisota/set/pull/2703" />

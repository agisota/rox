## Улучшенное представление чата — теперь общедоступно <PRBadge url="https://github.com/agisota/set/pull/1876" /> <PRBadge url="https://github.com/agisota/set/pull/1973" /> <PRBadge url="https://github.com/agisota/set/pull/1983" />

Переработанный интерфейс чата теперь общедоступен для всех пользователей. Раньше он был в раннем доступе, а новое представление чата отличается обновлёнными визуалами вызовов инструментов со сворачиваемыми панелями, улучшенной отрисовкой diff'ов и более чистыми представлениями файлов. Собственные инструменты MCP в Rox — задачи, рабочие пространства и сессии агентов — теперь отображаются как богатые интерактивные карточки прямо в чате. Rox Chat также доступен как полноценная опция агента с собственной конфигурацией запуска и иконкой.

- Обновлённые визуалы вызовов инструментов со сворачиваемыми панелями вывода и встроенными иконками статуса
- Богатые карточки интерфейса для вызовов инструментов Rox MCP (задачи, рабочие пространства, сессии агентов)
- Rox Chat доступен как встроенный агент наряду с Claude, Codex и другими
- Отрисовка markdown для вопросов агента с оптимистичным UX ответов

## Открытие worktree с промптом <PRBadge url="https://github.com/agisota/set/pull/1882" /> <PRBadge url="https://github.com/agisota/set/pull/1912" />

Модальное окно нового рабочего пространства переработано со сценарием отправки агенту. При создании worktree ты теперь можешь написать промпт, который отправляется прямо агенту в новом рабочем пространстве — опиши, что хочешь построить, и агент начинает работать, как только рабочее пространство готово. В модальном окне также есть селектор проектов, сценарий импорта для клонирования репозиториев и автоматически сгенерированные имена веток на основе твоего промпта.

![Открытие worktree с промптом](/changelog/ws-prompt.png)

- Напиши промпт задачи при создании нового рабочего пространства
- Агент сразу начинает работать в новом worktree
- Автоматически сгенерированные имена веток из содержимого промпта
- Подсказка горячей клавиши (Cmd/Ctrl+Enter) на кнопке создания

## Запуск проекта из шаблона <PRBadge url="https://github.com/agisota/set/pull/1913" />

Новые пользователи теперь видят многошаговый мастер настройки при первом открытии проекта. Онбординг автоматически определяет env-файлы и пакетные менеджеры, уже присутствующие в репозитории, проводит через запуск скриптов настройки и даёт ссылки на нужную документацию. Сценарий нового проекта также включает опцию шаблона Next.js Chatbot для быстрого старта проекта с готового стартера.

![Запуск проекта из шаблона](/changelog/template.png)

- Многошаговый мастер с определением скриптов настройки
- Автоопределение env-файлов и пакетных менеджеров
- Шаблон проекта Next.js Chatbot через vercel/chatbot
- Ссылки на docs на каждом шаге настройки

## Улучшения

- **Интерфейс промптов агента Mastra** — вопросы агента, запросы на одобрение и планы теперь появляются как интерактивные встроенные промпты в чате с отрисовкой markdown и оптимистичным UX ответов <PRBadge url="https://github.com/agisota/set/pull/1933" /> <PRBadge url="https://github.com/agisota/set/pull/1970" />
- **Сохранение выбора модели чата** — выбранная тобой модель теперь запоминается между сессиями и перезапусками приложения <PRBadge url="https://github.com/agisota/set/pull/1956" />
- **Встроенные инструменты Rox MCP** — собственные инструменты MCP в Rox подключены как встроенные дополнения в чате, ручная настройка не нужна <PRBadge url="https://github.com/agisota/set/pull/1858" />
- **Поддержка API-ключа Anthropic** — теперь ты можешь указать собственный API-ключ Anthropic для сессий чата Mastra <PRBadge url="https://github.com/agisota/set/pull/1905" />
- **Контекстное меню панели по правому клику** — клик правой кнопкой по панели чата или браузера теперь открывает контекстное меню с частыми действиями <PRBadge url="https://github.com/agisota/set/pull/1886" />
- **Открытие панели изменений из правок файлов** — когда агент редактирует файл в чате, ты можешь перейти прямо к панели изменений, чтобы просмотреть diff <PRBadge url="https://github.com/agisota/set/pull/1868" />
- **Cmd+Shift+W закрывает активную вкладку** — новая горячая клавиша для закрытия той вкладки панели, что в фокусе <PRBadge url="https://github.com/agisota/set/pull/1860" />
- **Монитор ресурсов включён по умолчанию** — монитор ресурсов в системном трее теперь включён для всех пользователей из коробки <PRBadge url="https://github.com/agisota/set/pull/1949" />
- **Diff'ы файлов через diffs.com** — diff'ы инструмента редактирования теперь по умолчанию открываются в diffs.com для более чистого опыта просмотра <PRBadge url="https://github.com/agisota/set/pull/1859" />
- **Инструмент чтения файла свёрнут по умолчанию** — результаты чтения файлов начинаются свёрнутыми, чтобы уменьшить шум в чате <PRBadge url="https://github.com/agisota/set/pull/1895" />
- **Windsurf IDE в меню «Открыть в»** — Windsurf теперь доступен в выпадающем списке внешних редакторов «Открыть в» <PRBadge url="https://github.com/agisota/set/pull/1966" />
- **Нативная интеграция шелла** — интерактивные терминалы теперь по умолчанию используют нативную интеграцию шелла для лучшей обработки PATH и окружения <PRBadge url="https://github.com/agisota/set/pull/1987" />
- **Переключатель месячного/годового биллинга** — Pro-пользователи могут переключаться между месячным и годовым биллингом прямо со страницы планов <PRBadge url="https://github.com/agisota/set/pull/1977" />
- **Улучшения производительности чата** — заменили агрессивный поллинг более умной стратегией обновлений, снизив нагрузку на полосу прокрутки и повысив скорость отрисовки <PRBadge url="https://github.com/agisota/set/pull/1872" />

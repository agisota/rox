## Расстановка вкладок терминала <PRBadge url="https://github.com/agisota/set/pull/980" />

- Включили переупорядочивание вкладок перетаскиванием для удобной организации рабочего пространства
- Показываем подсказки по навигации с клавиатуры в тултипах соседних вкладок
- Автоматически генерируем имена веток из названий рабочих пространств при их создании

## Поддержка docs и блога <PRBadge url="https://github.com/agisota/set/pull/974" />

- Переработали сайдбар docs с упрощённой структурой разделов и пунктов
- Обновили логотип в навигационной панели, чтобы он вёл на маркетинговый URL
- Добавили dotenv для загрузки env-переменных во время разработки
- Добавили overscroll-none, чтобы предотвратить поведение pull-to-refresh
- Починили маркетинговый редирект, чтобы он использовал env-переменную для URL docs

## Поддержка терминала Ghostty <PRBadge url="https://github.com/agisota/set/pull/950" />

Добавили интеграцию приложения Ghostty в возможности открытия файлов десктопного приложения. Пользователи теперь могут выбрать Ghostty из списка доступных приложений при использовании функции «Открыть в», чтобы открывать файлы в предпочитаемом терминальном приложении.

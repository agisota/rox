## Переработка сайдбара <PRBadge url="https://github.com/agisota/set/pull/601" />

- Поддержка аватаров GitHub в миниатюрах проектов
- Сворачиваемый сайдбар рабочих пространств с компактным режимом «только иконки» и улучшенным поведением переключения/сворачивания
- Более насыщенные заголовки проектов и компактные миниатюры
- Обновлены раскладки элементов сайдбара и списков, иконки, отступы и размеры во многих панелях

## Страница рабочих пространств <PRBadge url="https://github.com/agisota/set/pull/599" />

- Добавлен список рабочих пространств с поиском и фильтрацией, сгруппированными заголовками проектов и строкой на каждое рабочее пространство
- Новый интерфейс строки рабочего пространства показывает статус, diff'ы, временные метки и состояния переоткрытия/открытия
- Заголовки проектов теперь предлагают контекстное меню: открыть в Finder, настройки проекта, закрыть проект
- Заголовок сайдбара рабочих пространств стал кликабельным переключателем, а список рабочих пространств автоматически закрывается при выборе

## Индикаторы статуса рабочих пространств <PRBadge url="https://github.com/agisota/set/pull/588" />

- Добавили трёхцветные индикаторы статуса рабочих пространств, показывающие жизненный цикл агента (янтарный = работает, красный = нужно разрешение, зелёный = готово к ревью)
- Интеграция с хуками Claude Code и плагином OpenCode для обновления статуса в реальном времени
- Поведение по клику: ревью → ожидание (подтверждено), разрешение → работает (предполагается, что разрешение выдано)

## Статус PR в представлении изменений <PRBadge url="https://github.com/agisota/set/pull/608" />

Добавили индикатор статуса PR (иконка + номер) в заголовок представления изменений со ссылкой на GitHub. Цветовая кодировка иконок PR в зависимости от состояния (зелёный — открыт, фиолетовый — смержен, красный — закрыт, приглушённый — черновик).

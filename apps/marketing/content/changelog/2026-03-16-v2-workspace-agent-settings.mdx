## Настраиваемые параметры агентов <PRBadge url="https://github.com/agisota/set/pull/2495" />

Поведение агентов теперь настраивается из интерфейса настроек десктопа. Ты можешь переопределять встроенные пресеты, делать предпросмотр шаблонов промптов и переиспользовать одни и те же определения агентов везде, где запускаешь работу, не редактируя конфигурационные файлы вручную.

- Настраивай модель, метки, описания, команды и шаблоны промптов с одной страницы настроек
- Изменения сохраняются сразу и могут быть сброшены к значениям по умолчанию
- Те же общие пресеты агентов теперь питают сценарии создания рабочих пространств и запуска задач

## Упрощённое модальное окно рабочего пространства <PRBadge url="https://github.com/agisota/set/pull/2391" /> <PRBadge url="https://github.com/agisota/set/pull/2506" /> <PRBadge url="https://github.com/agisota/set/pull/2500" /> <PRBadge url="https://github.com/agisota/set/pull/2513" />

Модальное окно рабочего пространства теперь держит выбор проекта, промпты, изображения, PR'ы и issue в одном потоке, так что запускать рабочее пространство с полным контекстом быстрее.

![Модальное окно нового рабочего пространства](/changelog/new-workspace-modal-2026-03-16.png)

- Сценарий создания с приоритетом промпта и выбором проекта прямо в модальном окне
- Аватары проектов, детали владельца GitHub и встроенное `Add from GitHub` в селекторе
- Текст промпта, прикреплённые изображения, связанные PR'ы, связанные issue и выбранный агент — всё живёт в одном упрощённом потоке
- Текст промпта, вложения и выбранный агент сохраняются, когда ты закрываешь и снова открываешь модальное окно
- Предпросмотр префикса ветки, лучшее усечение длинных имён и поиск PR по вставленному URL GitHub

## Улучшения вкладок и панелей <PRBadge url="https://github.com/agisota/set/pull/2488" /> <PRBadge url="https://github.com/agisota/set/pull/2487" />

Раскладки десктопа теперь гибче. Ты можешь объединить целую вкладку с раскладкой другой панели перетаскиванием и переименовать заголовки панелей на месте, чтобы держать плотные рабочие пространства в порядке.

![Перетаскивание вкладки в панель](/changelog/drag-tab.png)

- Перетащи вкладку верхнего уровня в зоны сплита другой вкладки, чтобы объединить её слева, справа, сверху или снизу
- Исходные вкладки чисто удаляются после объединения, сохраняя принадлежность панелей и состояние раскладки
- Дважды кликни по заголовкам панелей, чтобы переименовать их прямо на месте

## Улучшения

- **Сайдбар рабочих пространств V2** — рабочие пространства сгруппированы по проектам в переработанном сайдбаре, а более новый сценарий с приоритетом проекта напрямую питает эту структуру <PRBadge url="https://github.com/agisota/set/pull/2399" />
- **Пилюли связанных issue Linear** — issue Linear, упомянутые в задачах агента, теперь отображаются как встроенные пилюли с названием и статусом issue, так что ты можешь ссылаться на задачи в чате, не покидая контекст <PRBadge url="https://github.com/agisota/set/pull/2383" />
- **Горячая клавиша для уравнивания сплитов панелей** — нажми `Cmd+Shift+0`, чтобы мгновенно сбалансировать все сплиты панелей до равной ширины <PRBadge url="https://github.com/agisota/set/pull/2381" />
- **Селектор шрифтов с поиском** — селектор шрифтов в настройках теперь выпадающий список с поиском вместо обычного текстового поля <PRBadge url="https://github.com/agisota/set/pull/2458" />
- **Буфер прокрутки терминала увеличен до 5к строк** — больше истории сохраняется в сессиях терминала <PRBadge url="https://github.com/agisota/set/pull/2480" />
- **Серверный поиск веток/PR/issue** — селекторы веток, PR и issue теперь ищут на стороне сервера и используют пагинацию для больших репозиториев <PRBadge url="https://github.com/agisota/set/pull/2442" />
- **Темы десктопа применяются к редактору и diff'ам** — активная тема десктопа теперь распространяется на редактор CodeMirror и представления diff'ов <PRBadge url="https://github.com/agisota/set/pull/2404" />
- **PR'ы отсортированы по недавнему обновлению** — списки PR в модальном окне рабочего пространства теперь сортируются по времени обновления <PRBadge url="https://github.com/agisota/set/pull/2406" />
- **Годовая цена показана как месячный эквивалент** — страница планов теперь отображает стоимость в месяц для годовых подписок <PRBadge url="https://github.com/agisota/set/pull/2396" />
- **Android Studio добавлен в выпадающий список IDE** <PRBadge url="https://github.com/agisota/set/pull/2428" />
- **Доступна OpenAI GPT-5.4** — обновлённый список моделей OpenAI заменяет GPT-5.2 на GPT-5.4 <PRBadge url="https://github.com/agisota/set/pull/2497" />
- **Новая вкладка как режим выполнения пресета по умолчанию** <PRBadge url="https://github.com/agisota/set/pull/1817" />

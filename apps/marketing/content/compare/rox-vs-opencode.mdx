OpenCode — это агент для кода: единственный AI-ассистент, который читает, пишет и отлаживает код в твоём терминале. Rox — это local-first десктопное рабочее пространство, которое запускает множество агентов кода, включая OpenCode, параллельно в изолированных Git worktree. OpenCode — это агент. Rox — это слой оркестрации и ревью вокруг множества агентов, и многие разработчики используют их вместе.

## Что такое Rox?

Rox — это local-first десктопное рабочее пространство для AI-агентов кода. Оно запускает Claude Code, OpenCode, Codex, Aider, Copilot, Cursor Agent, Gemini CLI, Rox Chat и другие агентские воркфлоу внутри изолированных Git worktree с постоянными сессиями терминала. Вокруг этого ядра добавлены встроенный редактор diff/файлов, панель чата, внутренний браузер для документации и dev-серверов, управление портами и инструменты MCP. Ты можешь делать ревью прямо в Rox или перейти в VS Code, Cursor, Windsurf, JetBrains или Xcode.

---

## Что такое OpenCode?

OpenCode — это open-source AI-агент для кода, созданный для терминала командой SST (95 000+ звёзд на GitHub). Ты описываешь, что хочешь, на естественном языке, и AI читает твою кодовую базу, пишет код, выполняет команды и изменяет файлы внутри TUI на Bubble Tea, написанного на Go. Его выдающаяся черта — гибкость моделей: 75+ провайдеров, включая Anthropic, OpenAI, Google, AWS Bedrock, OpenRouter и локальные модели через Ollama. Он включает интеграцию с LSP, поддержку MCP и интеграцию с GitHub Actions.

## Ключевые отличия

### Оркестратор против агента

OpenCode общается с AI-моделями, читает твой код и вносит изменения. Rox — это рабочее пространство вокруг агентов: оно запускает инструменты вроде OpenCode параллельно, каждый в своём изолированном окружении, а затем накладывает управление задачами, чат, превью в браузере и инструменты ревью. Запусти OpenCode внутри Rox, чтобы получить гибкость моделей на сессию и параллельное выполнение по сессиям.

### Последовательно против параллельно

С одним только OpenCode ты работаешь по одной задаче за раз: промпт, ревью, итерация, следующая задача. С Rox ты назначаешь задачи нескольким агентам одновременно — один пишет тесты, другой рефакторит API, третий чинит ошибки линта — каждый на своей ветке. Ты делаешь ревью diff по мере их завершения и мержишь, когда доволен.

### Изоляция кода

OpenCode изменяет файлы прямо в директории твоего проекта, если ты сам не настроишь изоляцию. Rox создаёт отдельный Git worktree на каждую задачу, давая каждому агенту изолированную копию репозитория. Именно это делает безопасным запуск 10 агентов сразу — ни один агент не может испортить работу другого.

### Поддержка моделей и провайдеров

OpenCode поддерживает 75+ провайдеров и позволяет переключаться посреди сессии. Rox наследует то, что поддерживают его агенты — запусти OpenCode внутри Rox, и ты получишь все 75+ провайдеров; запусти Claude Code, и ты получишь Claude.

## Цена

Rox предлагает бесплатный тариф и Pro за $20/место/месяц плюс то, что провайдеры твоих агентов берут за использование API. OpenCode предлагает Zen (оплата по факту по цене провайдера, $20 стартового кредита) и OpenCode Black ($200/месяц за доступ ко всем моделям).

## Что выбрать тебе?

**Выбирай OpenCode, если ты:**
- Хочешь единственного мощного AI-агента кода с максимальной гибкостью моделей
- Предпочитаешь интерактивное сотрудничество с AI-ассистентом в реальном времени
- Хочешь open-source альтернативу Claude Code без привязки к провайдеру
- Нуждаешься в контроле расходов по разным моделям через оплату по себестоимости в Zen

**Выбирай Rox, если ты:**
- Уже используешь CLI-агентов кода и хочешь масштабировать использование горизонтально
- Нуждаешься в запуске 5-10 агентов параллельно по разным задачам
- Хочешь изоляцию через Git worktree, чтобы агенты никогда не конфликтовали друг с другом
- Работаешь над большими кодовыми базами, где параллельное выполнение экономит часы

**Используй оба** для лучшего от каждого: OpenCode как агент, Rox как оркестратор. Каждая задача Rox запускает свою сессию OpenCode в своём worktree — гибкость моделей на сессию, параллельное выполнение по всем из них.

## Часто задаваемые вопросы

### Заменяет ли Rox собой OpenCode?

Не совсем. У Rox теперь есть встроенный чат, инструменты MCP, превью в браузере и ревью файлов, но его главная задача по-прежнему — координировать агентов и worktree. OpenCode — это агент. Rox — это слой, который позволяет безопасно запускать множество сессий OpenCode или смешивать OpenCode с другими агентами на одном репозитории.

### Можно ли запустить OpenCode внутри Rox?

Да. Каждая задача Rox запускает свою сессию OpenCode в своём worktree и ветке, давая тебе гибкость моделей на сессию плюс изоляцию и параллелизм по сессиям.

### Чем OpenCode отличается от Claude Code?

Оба — терминальные AI-агенты для кода. OpenCode — open source (MIT) и поддерживает 75+ провайдеров; Claude Code — проприетарный и привязан к Anthropic. На одной и той же базовой модели качество кода сопоставимо. OpenCode предлагает гибкость по провайдерам; Claude Code предлагает более тесную оптимизацию под Anthropic.

Claude Code — это AI-агент для кода от Anthropic, доступный как в терминале, так и в десктопном приложении. Rox — это local-first десктопное рабочее пространство, которое умеет запускать несколько сессий Claude Code, других агентов и Rox Chat параллельно в изолированных Git worktree. Они дополняют друг друга: Claude Code — это агент, а Rox — слой оркестрации и ревью вокруг множества агентов.

## Что такое Rox?

Rox — это local-first десктопное рабочее пространство для AI-агентов кода. Оно запускает Claude Code, Codex, OpenCode, Aider, Copilot, Cursor Agent, Gemini CLI, Rox Chat и другие агентские воркфлоу внутри изолированных Git worktree с постоянными сессиями терминала. Вокруг этого ядра добавлены встроенный редактор diff/файлов, панель чата, внутренний браузер для документации и dev-серверов, управление портами и инструменты MCP. Ты можешь делать ревью прямо в Rox или перейти в VS Code, Cursor, Windsurf, JetBrains или Xcode. Открытый исходный код под Elastic License 2.0 (ELv2).

---

## Что такое Claude Code?

Claude Code — это официальный агент для кода от Anthropic. CLI по-прежнему остаётся основным способом работы, но теперь Anthropic также документирует десктопное приложение для macOS и Windows, если тебе нужен Claude Code без постоянной жизни в терминале. Под капотом это всё тот же продукт: ты описываешь, что хочешь, Claude читает твою кодовую базу, пишет код, выполняет команды и итерирует по задаче, используя модели Anthropic.

## Ключевые отличия

### Оркестратор против агента

Claude Code — это агент: он общается с моделями Claude, читает твой код и вносит изменения. Rox — это рабочее пространство для оркестрации: оно запускает множество инстансов Claude Code параллельно, каждый в своём изолированном окружении, а затем даёт тебе чат, превью в браузере, ревью diff/файлов и воркфлоу с подключением через MCP вокруг них. Claude Code делает работу; Rox масштабирует и организует её.

### Один продуктовый интерфейс против множества изолированных сессий

С одним только Claude Code — будь то из CLI или десктопного приложения — ты по-прежнему работаешь в основном над одной задачей за раз. Можно открыть несколько вкладок или окон, но настраивать изоляцию и отслеживать получившиеся репозитории, ветки, diff и превью придётся вручную. Rox автоматизирует весь этот цикл: создаёшь задачу — и он поднимает worktree, создаёт ветку, запускает Claude Code и держит весь интерфейс ревью в одном месте.

### Автоматическая изоляция

Запуская Claude Code напрямую, ты позволяешь ему менять файлы в текущей директории. Если запустить два инстанса Claude Code в одной директории, они перезапишут изменения друг друга. Rox создаёт отдельный Git worktree на каждую задачу, давая каждому инстансу Claude Code изолированную копию репозитория. Именно это делает безопасным запуск 10+ агентов сразу.

### Гибкость в выборе агента

Claude Code работает только с моделями Anthropic. Rox всё равно, каким агентом ты пользуешься — запускай Claude Code для сложных рефакторингов, Codex для чётко определённых задач, Aider для итеративных изменений или комбинируй их под каждую задачу. Если завтра выйдет агент получше, ты сможешь использовать его в Rox сразу же, не меняя свой воркфлоу.

### Сохранность сессий

Claude Code даёт тебе сами агентские интерфейсы. Rox добавляет операционный слой вокруг них: списки задач, долгоживущие сессии, встроенное ревью diff/файлов, превью локальных приложений в браузере и воркфлоу чат/MCP, привязанный к тому же рабочему пространству. Это важно, когда ты координируешь множество задач по одному репозиторию, а не остаёшься внутри одной сессии Claude.

## Цена

Rox предлагает бесплатный тариф и Pro за $20/место/месяц. Claude Code сейчас требует аккаунт Pro, Max, Teams, Enterprise или Console, и Anthropic также документирует поддержку доступа через провайдеров — сервисы вроде Bedrock, Vertex AI и Foundry. Важно сравнивать не только цену, но и структуру: Claude Code даёт тебе агента, а Rox добавляет оркестрацию поверх.

## Что выбрать тебе?

**Выбирай один только Claude Code, если ты:**
- Работаешь над одной задачей за раз в режиме реального времени с AI
- Предпочитаешь воркфлоу с одним агентом и глубоким контекстом
- Не нуждаешься в запуске нескольких агентов параллельно
- Хочешь максимально простую настройку (просто установить и запустить)

**Выбирай Rox + Claude Code, если ты:**
- Хочешь запускать 5-10+ инстансов Claude Code параллельно по разным задачам
- Нуждаешься в автоматической изоляции через Git worktree, чтобы агенты никогда не конфликтовали
- Хочешь сессии, которые переживают сбои и перезапуски приложения
- Планируешь использовать другие агенты вместе с Claude Code (Codex, Aider, OpenCode)
- Работаешь над большими кодовыми базами, где параллельное выполнение экономит часы

Большинство пользователей Rox запускают Claude Code как основного агента. Rox не конкурирует с Claude Code — он делает Claude Code мощнее, добавляя параллелизм, изоляцию и сохранность.

## Часто задаваемые вопросы

### Заменяет ли Rox собой Claude Code?

Не совсем. У Rox теперь есть встроенный чат, инструменты MCP, превью в браузере и ревью файлов, но его главная задача по-прежнему — координировать агентов и worktree. Claude Code — это агент для кода. Rox — это слой, который позволяет безопасно запускать множество сессий Claude Code или смешивать Claude Code с другими агентами на одном репозитории.

### Можно ли использовать Claude Code без Rox?

Да. Claude Code работает сам по себе из CLI или десктопного приложения Anthropic. Rox добавляет ценность, когда ты хочешь запускать несколько сессий Claude Code параллельно с автоматической изоляцией — а это становится важным по мере того, как ты масштабируешься от одного агента к множеству.

### Сколько инстансов Claude Code можно запустить в Rox?

Столько, сколько позволяют твоя машина и лимиты API. Большинство разработчиков запускают 5-10 агентов одновременно. Демон Rox управляет конкурентностью, а его планирование по приоритетам гарантирует, что задача в фокусе получает все ресурсы.

### Меняет ли Rox то, как работает Claude Code?

Нет. Claude Code работает ровно так же, как в любом терминале. Rox просто предоставляет worktree, ветку и сессию терминала. Все функции Claude Code — использование инструментов, MCP-серверы, хуки, слэш-команды — работают как обычно.

### Open source ли Rox?

Нет. Rox — с открытым исходным кодом на GitHub под Elastic License 2.0 (ELv2). Claude Code тоже скорее с доступным исходным кодом, чем open source.

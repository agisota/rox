GitHub Copilot — это AI-напарник по программированию, который живёт внутри твоего редактора, предлагая инлайн-дополнения и чат. Rox — это local-first рабочее пространство, которое запускает множество AI-агентов кода параллельно, каждого в своём Git worktree. Они нацелены на разные воркфлоу: Copilot помогает тебе строку за строкой, пока ты печатаешь, тогда как Rox раздаёт автономных агентов работать над целыми задачами самостоятельно и даёт тебе рабочее пространство вокруг них.

## Что такое Rox?

Rox — это local-first десктопное рабочее пространство для AI-агентов кода. Оно запускает Claude Code, Codex, OpenCode, Aider, Copilot, Cursor Agent, Gemini CLI, Rox Chat и другие агентские воркфлоу внутри изолированных Git worktree с постоянными сессиями терминала. Вокруг этого ядра добавлены встроенный редактор diff/файлов, панель чата, внутренний браузер для документации и dev-серверов, управление портами и инструменты MCP. Ты можешь делать ревью прямо в Rox или перейти в VS Code, Cursor, Windsurf, JetBrains или Xcode. Открытый исходный код под Elastic License 2.0 (ELv2).

---

## Что такое GitHub Copilot?

GitHub Copilot — это AI-ассистент для кода от GitHub (Microsoft). Он интегрируется в твой редактор как расширение, предоставляя инлайн-дополнения кода (призрачный текст, пока ты печатаешь), панель чата для вопросов и генерации кода и режим Copilot Agent, который может вносить многофайловые изменения автономно. Copilot также предлагает облачный Coding Agent, который создаёт PR из GitHub Issues. Он использует несколько моделей, включая GPT-4o, Claude Sonnet и Gemini, доступных через инфраструктуру GitHub.

## Ключевые отличия

### Инлайн-ассистент против параллельного оркестратора

Copilot создан для опыта редактирования — он дополняет твой код, пока ты печатаешь, отвечает на вопросы в панели чата и может вносить изменения по файлам через режим Agent. Rox запускает множество автономных агентов одновременно, каждого над отдельной задачей в отдельном worktree, а затем добавляет чат, ревью файлов и превью в браузере вокруг этих задач. Copilot делает тебя быстрее в написании кода; Rox заставляет множество агентов работать параллельно, пока ты занят чем-то ещё.

### Дополнения против автономных задач

Основное взаимодействие Copilot — это инлайн-дополнение: ты печатаешь, он предлагает. Его режим Agent и Coding Agent могут справляться с более крупными задачами, но ядро опыта — это помощь в реальном времени, пока ты пишешь код. Rox раздаёт полностью автономные задачи — «отрефактори платёжный сервис», «добавь тесты для модуля авторизации» — и агенты работают самостоятельно, пока не закончат, а получившиеся diff и превью локальных приложений собираются внутри того же рабочего пространства.

### Интеграция в редактор против независимости от редактора

Copilot живёт внутри твоего редактора (VS Code, JetBrains, Neovim). Rox — это отдельный терминал, который работает вместе с любым редактором. Это значит, что Copilot может помогать с функциями, специфичными для редактирования (дополнения, инлайн-diff), тогда как Rox фокусируется на оркестрации и изоляции, не заботясь о том, каким редактором ты пользуешься.

### Гибкость в выборе модели и агента

Copilot поддерживает несколько моделей (GPT-4o, Claude, Gemini) через инфраструктуру GitHub. Rox поддерживает любого агента на основе CLI — и, как следствие, любые модели, которые эти агенты поддерживают. Разница в том, напрямую или через прокси: с Rox ты приносишь свои API-ключи и платишь провайдерам напрямую. С Copilot ты используешь доступ к моделям GitHub через их тарифные уровни.

### Приватность

Rox запускает рабочее пространство локально в Git worktree, а его кодовая база с открытым исходным кодом полностью пригодна для аудита. Copilot отправляет контекст кода на серверы GitHub/Microsoft для обработки. Copilot Business и Enterprise предлагают контроль над хранением данных, но код всё равно проходит через внешние серверы.

## Цена

Rox предлагает бесплатный тариф и Pro за $20/место/месяц плюс расходы на API твоих агентов. GitHub Copilot предлагает Free (ограниченный), Pro ($10/мес или $100/год), Business ($19/польз/мес) и Enterprise ($39/польз/мес). Цена Copilot включает доступ к моделям; цена Rox — нет (за расходы на API агентов ты платишь отдельно).

## Что выбрать тебе?

**Выбирай GitHub Copilot, если ты:**
- Хочешь инлайн-дополнения кода в реальном времени, пока печатаешь
- Предпочитаешь помощь AI, интегрированную прямо в твой редактор
- Работаешь в основном в VS Code или JetBrains
- Хочешь простого, недорогого AI-ассистента ($10/мес), который повышает скорость редактирования

**Выбирай Rox, если ты:**
- Запускаешь агентов кода на основе CLI и хочешь распараллелить работу по 10+ задачам
- Нуждаешься в автономных агентах, работающих над отдельными задачами, пока ты занят другим
- Хочешь гибкость в выборе агента и модели без привязки к вендору
- Нуждаешься в локальном выполнении и прямом контроле над провайдерами
- Работаешь над большими кодовыми базами, где параллельное выполнение экономит часы

**Используй оба** для лучшего от каждого: Copilot для инлайн-дополнений, пока пишешь код, и Rox — чтобы раздавать параллельных агентов для более крупных задач. Они не пересекаются — Copilot помогает писать код быстрее, Rox помогает шире масштабировать работу агентов.

## Часто задаваемые вопросы

### Заменяет ли Rox собой Copilot?

Нет. Rox — не ассистент инлайн-дополнений. Он даёт встроенный чат, редактирование diff/файлов, превью в браузере и передачу в IDE, но не пытается заменить живые подсказки Copilot, пока ты печатаешь. Используй Copilot для помощи в редактировании в реальном времени, а Rox — для параллельных автономных задач.

### Чем Coding Agent от Copilot отличается от Rox?

Coding Agent от Copilot работает в облачной ВМ и создаёт PR из GitHub Issues — по одной задаче за раз на репозиторий. Rox запускает множество агентов локально параллельно по Git worktree. Агент Copilot хостится в облаке и интегрирован с GitHub; подход Rox — локальный, не зависящий от агента и параллельный.

### Можно ли использовать Copilot внутри worktree Rox?

Да. Когда ты открываешь worktree Rox в VS Code, Copilot работает как обычно — он видит worktree как обычный git-репозиторий. Ты получаешь дополнения Copilot, пока делаешь ревью или редактируешь вывод агента.

### Open source ли Rox?

Нет. Rox — с открытым исходным кодом на GitHub под Elastic License 2.0 (ELv2). GitHub Copilot — с закрытым исходным кодом.
